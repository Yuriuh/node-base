### 查看编码

```
show variables like 'character_set\_%';
```

### 查看数据库

```
show databases;
```

### 创建数据库

```
CREATE DATABASE 数据库名;
```

### 使用数据库

```
use 数据库名;
```

### 查看数据库

```
show 数据库名;
```

### 删除数据库
DROP DATABASE 数据库名;

### 查看数据表

```
show 表名;
```

### 查看列信息

```
show columns from 表名;
```

### 查看索引

```
show index from 表名;
```

### 查看统计信息

```
show table status from 数据库名;
```
